CMPS 3500 MiniScheme Public Test Set

This folder contains the official public MiniScheme case files for the comparative implementation project.

Categories:
- core_*.scm: literals, arithmetic, primitive evaluation, if
- let_*.scm: lexical scope, bindings, shadowing
- lambda_*.scm: lambda expressions and application
- recursion_*.scm: define and recursion
- error_*.scm: required runtime error categories
- syntax_*.scm: required parse error cases
- addon_*.scm: public cases for the announced cond add-on
- challenge_public_*.scm: published challenge cases for the live demo

These files are the authoritative public cases for development, testing, and demo preparation.
Hidden grading and hidden demo challenge cases are not included here.

Expected outputs and error categories are listed in expected_results.txt.
